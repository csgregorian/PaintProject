Paint Project Documentation

Modules

pygame - The main graphics library used for this project.

os - Used to lock the display surface to the top left corner of the screen.

sys->exit - Safely ends the process after exiting the main loop.

random->randrange - Pseudorandom number generator to calculate the spray bottle tool.

time->sleep - Waits during the splash screen.

math->hypot - Helps find the distance between two points.

NumPy - Extensive number processing library: covnerts surfaces to arrays and vice versa for quick and complex operations.

SciPy->ndimage/signal - Used to process NumPy arrays for filters.

Note: If NumPy and/or SciPy cannot be imported, filters will be disabled.

colours->cmyk - Converts RGB to CMYK based on an arbitrary standard colour profile.

stamps->stamps - A dictionary of the images used for the stamp tool.
