Global Functions

Mouse Functions

tm() - Returns the location of the mouse on the screen.

cm() - Returns the location of the mouse relative to the canvas (adjusted by +100/+100).

Filter Functions

gaussianBlur(surf) - Applies SciPy gaussian_filter to a numpy array and then returns a surface.

sobel(surf) - Applies SciPy sobel filter to a numpy array and then returns a surface.

invert(surf) - Inverts all the colour values by XORing them with 2**32 - 1 (36 1s in binary). Because each pixel is represented as a 32-bit colour value, this works by changing all 1s to 0s, and all 0s to 1s.
