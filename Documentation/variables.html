Global Variables

Mouse Functions

tools - A dictionary that stores all the tool objects.

currentTool - A string that holds the key in tools for the current tool.

toolLoc/toolLoc2 - Tuples that stores the canvas mouse location used by the tools.

size - Current tool size (brush radius, line width, etc).

currentColour - Tuple that holds the current foreground colour value.

paletteHue - Tuple that holds the current hue to blit behind the saturation/luminosity grid.

colourLoc - Tuple that holds the current colour location on the Sat/Lum grid.

currentStamp - Integer representation of the current numpad key stamp.

lastClick - String that shows where the last mouseDown action occured.

displayInfo - Boolean that chooses whether to display the Properties or the Info pane.

newLayer - Default empty surface with dimensions of (1080, 660) and a transparency colourkey of (255, 0, 255).

layers - List of layers.

currentLayer - Active layer in layers.

states - A list storing all the states in history in the form (copy of layers, copy of currentLayer).

currentState - Active state in states.
